# אפיון: תורה 2048 — חמשת החומשים

**תאריך:** 2026-06-22
**פרויקט:** משחק עצמאי תחת Lemons Gaming, עם נוכחות בסילבוס ([[project-gaming-syllabus]])

## רקע ומטרה

יש משחק קיים `bamidbar-2048.html` — וריאנט 2048 שבו האריחים הם **פרשות ספר במדבר**
לפי הסדר (במדבר→נשא→…→מסעי); מאחדים שני אריחים זהים כדי להתקדם לפרשה הבאה,
והמטרה להגיע לפרשה האחרונה. המשתמש רוצה **אתר אחד שמכיל את כל חמשת החומשים**
(בראשית, שמות, ויקרא, במדבר, דברים) — שאפשר לשחק בכולם, עם **בידול צבעוני** בין החומשים.
אתר **עצמאי** תחת Lemons Gaming, ועם **נוכחות בסילבוס** (קישור).

## החלטות (אושרו עם המשתמש)

1. **אחסון:** ריפו עצמאי חדש `Lemons-Gaming/torah-2048` → `https://lemons-gaming.github.io/torah-2048/`
   (זהה לדפוס המשחקים הקיימים: tap-and-pop, break-the-word).
2. **תוכן:** כל הפרשות של כל חומש, אותנטי (54 פרשות סה"כ). המטרה בכל חומש =
   להגיע לפרשה האחרונה שלו; אפשר להמשיך לשחק אחרי ניצחון.
3. **בידול צבעוני:** לכל חומש פלטה משלו (אושרה במוקאפ).
4. **מיקום בסילבוס:** כרטיס/צ'יפ בעולם 8 (מיומנויות נוספות).

## ארכיטקטורה

קובץ יחיד `index.html` בריפו החדש (בסגנון קובץ-יחיד של המשחקים הקיימים), עם שני מסכים:

### א. מסך תפריט (`בחרו חומש`)
5 כרטיסים, אחד לכל חומש, כל אחד בצבע ה-accent שלו, מציג: שם החומש, מספר פרשות,
וטווח (מהפרשה הראשונה עד האחרונה). לחיצה → מסך משחק של אותו חומש.

### ב. מסך משחק
**מנוע המשחק של `bamidbar-2048.html` בשימוש חוזר מילה-במילה**, מפורמט לפי החומש הנבחר:
- האריחים = פרשות החומש לפי הסדר; שני אריחים זהים מתאחדים לפרשה הבאה.
- סולם פרשות (ladder), ניקוד + שיא, אוברליי ניצחון/הפסד, "המשך לשחק".
- מטרה = הפרשה האחרונה של החומש (אבן היעד, עם זוהר).
- כפתור **"← לחומשים"** חוזר לתפריט.
- קלט: החלקה (נייד) + מקשי חיצים/WASD (מחשב) — כמו במשחק הקיים.

מה שמשתנה מהמשחק הקיים: `NAME`/`COLORS`/`ORDER`/`MAX` הופכים לנתונים פר-חומש
שנגזרים ממבנה `BOOKS`; טעינת מסך משחק מאתחלת את המנוע עם החומש הנבחר.

### ג. נתוני החומשים (`BOOKS`)
מערך של 5 אובייקטים. פרשות (לפי הסדר השנתי):
- **בראשית** (12): בראשית, נח, לך לך, וירא, חיי שרה, תולדות, ויצא, וישלח, וישב, מקץ, ויגש, ויחי — יעד 4096
- **שמות** (11): שמות, וארא, בא, בשלח, יתרו, משפטים, תרומה, תצוה, כי תשא, ויקהל, פקודי — יעד 2048
- **ויקרא** (10): ויקרא, צו, שמיני, תזריע, מצורע, אחרי מות, קדושים, אמור, בהר, בחקתי — יעד 1024
- **במדבר** (10): במדבר, נשא, בהעלותך, שלח, קרח, חקת, בלק, פינחס, מטות, מסעי — יעד 1024
- **דברים** (11): דברים, ואתחנן, עקב, ראה, שופטים, כי תצא, כי תבוא, נצבים, וילך, האזינו, וזאת הברכה — יעד 2048

ערכי האריחים = חזקות 2 החל מ-2 (10/11/12 אריחים → יעד 1024/2048/4096). הלוח נשאר 4×4.

## פלטות צבע (אושרו במוקאפ)

בסיס קלף (parchment) משותף; לכל חומש רמפת accent מבהיר→מתכהה לאורך הסולם.
המרה: רמפה נוצרת ע"י אינטרפולציה לינארית בין `start` ל-`end` על-פני מספר הפרשות
(לבמדבר — רמפה אצורה קיימת, זהב→אינדיגו). צבע טקסט: אבן היעד = זהב `#f4cd7e`;
אחרת אם בהירות הרקע < 0.55 → `#fff7ea`, אחרת `#4a3414`.

- **בראשית** ירוק — accent `#2f6b46`, start `#e7e6c8`, end `#173b29`
- **שמות** תכלת — accent `#2b6f88`, start `#dde9e6`, end `#13314a`
- **ויקרא** אדום — accent `#a23d3f`, start `#eedcc8`, end `#5c1c22`
- **במדבר** זהב→אינדיגו — accent `#9c6b34`, רמפה אצורה: `['#e9ddc4','#e1c995','#dcab66','#d28a4a','#c06f3e','#a9543f','#8f4350','#6f3c63','#403a6b','#23264e']`
- **דברים** סגול — accent `#5a3f7a`, start `#e4dbe8`, end `#2c2047`

## שמירת נתונים

שיא לכל חומש בנפרד דרך **`localStorage`** (מפתח כמו `torah2048-best-<bookId>`).
מחליף את `window.storage` של המשחק הקיים (שלא קיים באתר עצמאי). עטוף ב-try/catch.

## נוכחות בסילבוס

כרטיס/צ'יפ בעולם 8 (`data-world="8"`) ב-`index.html` של הסילבוס, מקשר ל-
`https://lemons-gaming.github.io/torah-2048/` (`target="_blank"`), בדפוס הצ'יפים הקיים.
