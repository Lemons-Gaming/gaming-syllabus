# Torah 2048 (Five Books) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a standalone Hebrew/RTL "תורה 2048" site containing all five books of the Torah as color-differentiated 2048-style games (tiles = the book's parashot in order), deployed to a new `Lemons-Gaming/torah-2048` repo, and linked from World 8 of the syllabus.

**Architecture:** A single self-contained `index.html` reusing the game engine from the existing `bamidbar-2048.html` verbatim, but parameterized by a `BOOKS` data structure. A menu screen lists the 5 books (each in its palette); selecting one initializes the engine with that book's parashot, color ramp, ladder, and win target, with per-book best score in `localStorage`.

**Tech Stack:** Static HTML/CSS/vanilla JS (no build, no test framework). Google Fonts. Deployed via `gh` to GitHub Pages. Verification is manual/browser (no unit tests).

## Global Constraints

- Language/direction: Hebrew, `dir="rtl"`, `lang="he"`. All UI copy Hebrew.
- The game engine (move/merge/spawn/render/input/overlays) is REUSED VERBATIM from
  `bamidbar-2048.html` (in the syllabus repo root) — do not redesign the 2048 mechanic.
- 4×4 board. Tiles are parashah names; merging two identical advances to the next
  parashah; win = reaching the book's LAST parashah; "keep playing" allowed after.
- Fonts: Frank Ruhl Libre (serif) + Heebo (sans), via the same Google Fonts link as `bamidbar-2048.html`.
- Parchment base theme shared; each book overrides the tile color ramp + accent.
- Per-book best score via `localStorage`, key `torah2048-best-<bookId>` (replaces the
  old `window.storage`). Wrap access in try/catch.
- 5 books with EXACT parashot (annual order) and win targets:
  - genesis בראשית (12) → 4096 · exodus שמות (11) → 2048 · leviticus ויקרא (10) → 1024 · numbers במדבר (10) → 1024 · deut דברים (11) → 2048
- Palettes (exact): genesis accent `#2f6b46` ramp `#e7e6c8`→`#173b29`; exodus accent `#2b6f88` ramp `#dde9e6`→`#13314a`; leviticus accent `#a23d3f` ramp `#eedcc8`→`#5c1c22`; numbers accent `#9c6b34` curated ramp `['#e9ddc4','#e1c995','#dcab66','#d28a4a','#c06f3e','#a9543f','#8f4350','#6f3c63','#403a6b','#23264e']`; deut accent `#5a3f7a` ramp `#e4dbe8`→`#2c2047`.
- Tile text color: destination (last) tile = `#f4cd7e`; else luminance(bg)<0.55 → `#fff7ea`, else `#4a3414`.
- The standalone site lives in a NEW repo `Lemons-Gaming/torah-2048` → `https://lemons-gaming.github.io/torah-2048/`.
- Deploy actions (create repo, push, enable Pages, push syllabus card to `main`) are approval-gated — do not run them without explicit user confirmation.

```javascript
function hex2rgb(h){h=h.replace('#','');return [parseInt(h.slice(0,2),16),parseInt(h.slice(2,4),16),parseInt(h.slice(4,6),16)];}

function rampColors(start,end,n){const A=hex2rgb(start),B=hex2rgb(end),o=[];for(let i=0;i<n;i++){const t=n<=1?0:i/(n-1);o.push(rgb2hex([A[0]+(B[0]-A[0])*t,A[1]+(B[1]-A[1])*t,A[2]+(B[2]-A[2])*t]));}return o;}
```

```javascript
let currentBook = null;

function deriveBook(book){
  const ps = book.parashot, cols = bookColors(book);
  ORDER = ps.map((_,i)=>Math.pow(2,i+1));      // [2,4,8,...]
  MAX   = ORDER[ORDER.length-1];
  NAME = {}; COLORS = {};
  ORDER.forEach((v,i)=>{ NAME[v]=ps[i]; COLORS[v]={bg:cols[i], fg:fgFor(cols[i], i===ps.length-1)}; });
}

function buildLadder(){
  const L = document.getElementById('ladder'); L.innerHTML='';
  ORDER.forEach((v,i)=>{
    const c=COLORS[v]; const chip=document.createElement('div');
    chip.className='chip'; chip.style.background=c.bg; chip.style.color=c.fg;
    chip.innerHTML='<span class="ord">'+(i+1)+'</span>'+NAME[v];
    L.appendChild(chip);
  });
}

function bestKey(){ return 'torah2048-best-'+currentBook.id; }
function loadBest(){ try{ best = parseInt(localStorage.getItem(bestKey())||'0',10)||0; }catch(e){ best=0; } bestEl.textContent=best; }
function saveBest(){ try{ localStorage.setItem(bestKey(), String(best)); }catch(e){} }

function startBook(book){
  currentBook = book;
  deriveBook(book);
  document.getElementById('book-name').textContent = book.name;
  document.getElementById('goal-text').textContent = 'המטרה: להגיע אל '+book.parashot[book.parashot.length-1]+' — סיום ספר '+book.name+'.';
```
